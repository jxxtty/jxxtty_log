### update 벌크성 쿼리
```java
@Override
public long changeEventStatus(List<Long> idList, EventStatus eventStatus) {
    return jpaQueryFactory
        .update(event)
        .set(event.eventStatus, eventStatus)
        .where(event.id.in(idList))
        .execute();
}
```

### delete 벌크성 쿼리
```java
@Override
public long deleteEvent(List<Long> idList) {
    return jpaQueryFactory
        .delete(event)
        .where(event.id.in(idList))
        .execute();
}
```
